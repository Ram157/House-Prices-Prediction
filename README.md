# House Prices Prediction

## Описание

Проект решает задачу регрессии: предсказание цены дома по его характеристикам.

## Данные

Используется Kaggle competition House Prices - Advanced Regression Techniques.

## Метрика

Основная метрика: RMSE на логарифме SalePrice.

## Подход

1. EDA
2. Обработка пропусков
3. Кодирование категориальных признаков
4. Feature engineering
5. Обучение нескольких моделей
6. Cross-validation
7. Создание submission-файла

## Модели

- Ridge Regression
- Lasso Regression
- Random Forest
- Gradient Boosting
- XGBoost / LightGBM, если добавишь
